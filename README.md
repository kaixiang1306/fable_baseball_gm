# 寶島職棒 GM（傳奇經理）

網頁版棒球賽季模擬經營遊戲，以台灣職棒文化為藍本：執掌一支職棒球隊的總經理，
帶領球隊打完上下半季、殺進台灣大賽，同時滿足老闆的目標——否則就等著被解雇。

## 遊戲特色

- **台灣六都原創球隊**：台北雷虎、桃園飛龍、新竹風獅、台中太陽、台南武士、高雄海鷹，
  各有配色、老闆性格與偏好（金錢／戰績／形象／士氣／耐心）
- **台式賽制**：上下半季各 60 場、半季冠軍進台灣大賽（七戰四勝）、延長 12 局和局制、
  季中北軍 vs 南軍明星賽（可觀看）
- **逐打席即時模擬引擎**：三振／保送／安打／全壘打／雙殺／盜壘／觸擊／高飛犧牲、
  投手體力與牛棚調度、勝敗投與救援成功判定
- **文字轉播＋臨場指揮**：逐打席觀看比賽（可調速／跳過），輪到自家比賽時可下達
  **代打、觸擊、敬遠、手動換投**等戰術指令，並即時顯示投手用球數與疲勞狀態
- **守備適性與失誤**：球員守非本職位置會折損守備值（打線編排顯示適性 %），
  比賽會出現守備失誤（記入個人失誤數，失誤跑者得分為非自責分）
- **完整 GM 經營**：
  - 打序調整、先發輪值、終結者設定、一二軍升降（26 人一軍）
  - **二軍賽事**：每日與一軍同步開打，獨立數據與戰績，年輕球員累積出賽加速成長
  - **傷兵系統**：比賽會產生傷勢（5～80 天），自動移出一軍並遞補，傷癒通知歸隊
  - **個人士氣**：受勝率、出賽機會、薪資合理性影響，回饋到場上表現與續約要價
  - **續約談判**：季中與自家球員談延長合約，談崩三次本季拒絕再談
  - 交易系統（AI 評估價值、薪資空間、傷勢與士氣，季中第 90 天截止）
  - 休賽季自由球員市場與新人選秀（3 輪、戰績反序）、AI 自動綁約主力
  - 財務系統：票房收入、薪資預算、年度損益
  - 老闆評鑑：依老闆偏好生成年度目標，未達標耗損信任度，歸零即解雇
- **球探系統與潛力迷霧**：潛力值不直接可見，只顯示球探估計範圍——
  初步報告可能高估（水貨）或低估（遺珠），派球探考察（每 5 天累積 1 點、
  選秀前加發）可逐級收窄範圍直到精確；自家球員教練團預設有詳細情報，
  選秀名單排序也只反映你掌握的情報
- **歷史與名人堂**：生涯數據累計、歷年冠軍與數據王紀錄、引退球星名人堂
- **球員個人頁面**：點擊任何球員名字開啟個人卡——能力條、本季一二軍成績、
  生涯通算、逐季紀錄表與 OVR 生涯曲線圖
- 自動存檔（localStorage），可跨次接續經營多個賽季
- **聯盟存檔匯出／匯入**：一鍵下載 JSON 存檔分享，匯入時可改選執掌球隊，
  支援多人輪流執掌同一聯盟（pass-and-play）

## 開發

```bash
npm install
npm run dev        # 開發伺服器
npm run build      # 產出 dist/（純靜態檔，可部署至任何靜態主機）
npm run calibrate  # 模擬一整季並輸出聯盟數據（引擎校準用）
npx tsx scripts/fullcycle.ts  # 多賽季完整循環測試
```

技術：React 18 + TypeScript + Vite + Zustand，純前端、無後端依賴。

## 程式結構

```
src/
  types.ts            核心型別（球員/球隊/聯盟/賽程/歷史）
  data/teams.ts       六隊設定與老闆人設
  data/names.ts       台灣風格姓名與洋將譯名池
  engine/
    playerGen.ts      球員生成、綜合能力、薪資估算
    positions.ts      守備位置適性矩陣
    scouting.ts       球探系統與潛力迷霧
    sim.ts            LiveGame 逐打席即時引擎（代打/觸擊/敬遠/換投/失誤）
    league.ts         聯盟建立、賽程生成、陣容自動編成
    season.ts         賽季推進、戰績、台灣大賽、歷史紀錄、老闆評鑑
    farm.ts           二軍賽事
    allstar.ts        明星賽（北軍 vs 南軍）
    rosterOps.ts      傷兵遞補與 AI 陣容管理
    contracts.ts      續約談判與士氣漂移
    trade.ts          交易價值與 AI 評估
    offseason.ts      自由市場、選秀、球員成長、名人堂、新賽季
  store.ts            Zustand 狀態與 localStorage 存檔
  ui/                 各畫面（選隊/行事曆/轉播/陣容/交易/財務/歷史/選秀…）
```
