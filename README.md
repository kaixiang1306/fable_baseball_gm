# 寶島職棒 GM（傳奇經理）

網頁版棒球賽季模擬經營遊戲，以中華職棒為藍本：執掌一支職棒球隊的總經理，
帶領球隊打完上下半季、殺進台灣大賽，同時滿足老闆的目標——否則就等著被解雇。

## 遊戲特色

- **致敬中職六隊**：鼎信兄弟、統盛獅、樂添桃猿、福邦悍將、味泉龍、台鈦雄鷹，
  各有風格配色、老闆性格與偏好（金錢／戰績／形象／士氣／耐心）
- **中職賽制**：上下半季各 60 場、半季冠軍進台灣大賽（七戰四勝）、延長 12 局和局制
- **逐打席模擬引擎**：三振／保送／安打／全壘打／雙殺／盜壘／高飛犧牲、
  投手體力與牛棚調度、勝敗投與救援成功判定
- **文字轉播觀看模式**：可逐打席觀看比賽（可調速／跳過），也可快速模擬
- **完整 GM 經營**：
  - 打序調整、先發輪值、終結者設定、一二軍升降（26 人一軍）
  - 交易系統（AI 評估價值與薪資空間，季中第 90 天交易截止）
  - 休賽季自由球員市場與新人選秀（3 輪、戰績反序）
  - 球員隨年齡成長／衰退、合約年限、洋將制度
  - 財務系統：票房收入、薪資預算、年度損益
  - 老闆評鑑：依老闆偏好生成年度目標，未達標耗損信任度，歸零即解雇
- 自動存檔（localStorage），可跨次接續經營多個賽季

## 開發

```bash
npm install
npm run dev        # 開發伺服器
npm run build      # 產出 dist/（純靜態檔，可部署至任何靜態主機）
npm run calibrate  # 模擬一整季並輸出聯盟數據（引擎校準用）
npx tsx scripts/fullcycle.ts  # 多賽季完整循環測試
```

技術：React 18 + TypeScript + Vite + Zustand，純前端、無後端依賴。

## 程式結構

```
src/
  types.ts            核心型別（球員/球隊/聯盟/賽程）
  data/teams.ts       六隊設定與老闆人設
  data/names.ts       台灣風格姓名與洋將譯名池
  engine/
    playerGen.ts      球員生成、綜合能力、薪資估算
    sim.ts            逐打席比賽模擬引擎
    league.ts         聯盟建立、賽程生成、陣容自動編成
    season.ts         賽季推進、戰績、台灣大賽、老闆評鑑
    trade.ts          交易價值與 AI 評估
    offseason.ts      自由市場、選秀、球員成長、新賽季
  store.ts            Zustand 狀態與 localStorage 存檔
  ui/                 各畫面（選隊/行事曆/轉播/陣容/交易/財務/選秀…）
```
